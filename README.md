# AxiumOS

Axium Open Structure (AOS).

## AXIR development branches

- `bootstrap` contains the preserved C++ AXIR bootstrap implementation.
- `main` contains the AXIR specification and the self-hosting implementation path.

See `axir/AXIR_SPEC_v1.0.md` for the language specification and `axir/SELF_HOSTING.md` for the verified self-hosting plan and its required target-contract decision.
